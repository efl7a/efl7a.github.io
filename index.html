PTA Mobile App

An IOS/android app using React Native. This app allows school PTAs to use Firebase to store school information so the need for updates will be minimal. The app also uses Expo to send users notifications.

Come On In

A web app for schools to keep track of before/after school groups using RoR backend and JS frontend. It allows teachers to create study groups and students to sign up for them. It also allows administration to print out a master list of all students that are expected to be on campus before or after school hours.

GitHub Repository

This is a web app that allows users to search rezoning requests for Charlotte, NC. It also allows users to comment on those requests.
